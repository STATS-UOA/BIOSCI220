Using R

The latter weeks of this section will rely on the use of R. For this you'll need to use R on your own computer; I'd recommend using R through RStudio. Technically speaking R is the programming language and RStudio is, probably the most popular, an interface you can use to work with R. RStudio can be downloaded by following these instructions but remember you'll need to have R installed first (instructions here).

However, if you cannot install R & RStudio onto your own machine then I would reccommend using RStudio Cloud. Here you can run your analysis driectly from your browser; there is nothing to configure or install on your comupter! I would stress that if you have a workflow using R that you are already comfortable with there is no nee to learn yet another platform!

Topics

Nicole, in Module 1, will have introduced you to the programming language R and Nick, in Module 2, will have talked you through some of the fundamental concepts of statistical modelling. Now we're going to bring the two together and get some hands on experience! We will be covering five topics during this section of the course. These are

  • Experimental Design
  • Visualising and analysis of multivariate data
  • Hypothesis testing
  • Introduction to linear modelling
  • Modelling II

Below you'll find all the materials for this section of the course. Prior to each week's lab I will be uploading videos and extra materials to help with your assessed lab work. There are 3 main sections below:

  • Lab materials, where materials for each week's lab will be available as well as details for the required assessed lab work;
